# medicine_delivery_app(middle)

### Product Management:
* Browse and Manage Products: Users can effortlessly explore and handle a diverse selection of products.
* Intuitive Controls: Easy removal or modification of product quantities through user-friendly controls.
* Advanced Sorting: Enhanced sorting options allow users to organize products based on price and date added.
* Favorites Feature: Users can mark medicines as favorites for a personalized experience.
###Order Placement:
* Add to Cart: Conveniently add desired medicines to the cart from various drug stores.
* Seamless Order Submission: Streamlined process for submitting orders, including provision for email, phone number, and address details.
### Shopping Cart:
* Persistent Cart: Utilizes local storage to retain cart items, ensuring continuity across user sessions.
* Easy Modification: Users can effortlessly review and modify items in their cart, facilitating seamless order management.
